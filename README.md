  <h4>주요기능</h4>
	<P>1. 회원가입(이메일을 사용해 인증번호를 이용)</P>
  <P>2. 카카오api(OAuth2)를 이용한 회원가입</P>
  <P>3. 전시등록,관리 기능(관리자)</P>
  <P>4. 전시예약,수정,취소 기능</P>
  <P>5. 전시상세정보,리스트,검색 기능</P>
  <P>6. 게시판 글쓰기,수정,삭제 기능(이미지포함)</P>
  <P>7. 회원탈퇴</P>
